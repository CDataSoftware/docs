## 幅広いデータコネクティビティ

Embedded Cloud は、データソースとのリアルタイムなデータコネクティビティを実現するクラウドアプリケーションです。Embedded Cloud を使用することで、データを活用するあらゆるアプリケーションが、連携したデータソースのデータを読み書きできます。例としては次のようなカテゴリのツールが挙げられます：

- BI とデータ分析ツール
- ETL とデータ移行
- データウェアハウス
- データガバナンス
- マスターデータ管理
- データ統合

## リアルタイムなデータアクセス

Embedded Cloud は、データを活用するあらゆるアプリケーションで広くサポートされている標準インターフェースを使用して、エンタープライズSaaS、NoSQL、ビッグデータなどのデータソースへのリアルタイムのデータコネクティビティを実現します。

Embedded Cloud の主な特徴は次のとおりです：

- インストール不要、個々のドライバー設定不要、データベース不要
- 幅広いデータアプリケーションと互換性あり
- リアルタイムデータの読み、書き、更新が可能
- 高度なメタデータ検出機能
- オンプレミスからクラウド、クラウドからオンプレミス、クラウドからクラウドへの接続をサポート

## 集中管理型コネクティビティ

Embedded Cloud は、すべてのデータをクラウド上の一か所に統合します。この統合には次のようなメリットがあります：

- すべてのアカウントのクレデンシャルを一元管理できる。この機能により、IT 部門が組織内のさまざまなユーザーにクレデンシャルを配布し、管理する必要がなくなります。
- 権限のきめ細かな制御が可能。データソースごとに複数のアカウントを作成することなく、特定のユーザーの権限を制御できます。
- インテグレーションは、Embedded Cloud へのコネクションが1つあるだけで、すべてのデータソースにアクセスできる。

## 高機能なSQL サポート

- **文字列、日付、数値のSQL 関数**  
	50種類以上の関数からなるライブラリにより、カラムの値を操作して目的の結果を得ることができます。代表的な例としては正規表現、JSON、およびXML 処理用の関数があります。
- **包括的なクエリ処理**  
	Embedded Cloud は、CData Software のドライバーと同じエンタープライズクラスのクエリエンジンを採用しています。このエンジンは、必要に応じて追加のクライアント側の処理を実行してSUM、AVG、MAX、MIN などのデータ集計機能を追加し、データソースの機能を強化します。
- **エンタープライズクラスのセキュアなコネクティビティ**  
	Embedded Cloud には、すべてのクライアント - サーバー間接続でTLS / SSL データ暗号化などのエンタープライズレベルのセキュリティ機能が利用できます。
- **メタデータ検出**  
	Embedded Cloud は、データソースのメタデータ検出をサポートします。データ型は標準的なデータベース用の型として返されるため、BI、ETL、データ分析プラットフォームから簡単にデータを操作できます。
