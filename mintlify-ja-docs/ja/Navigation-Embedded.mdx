Click the **Users** tab to view a list of users:

<Frame>
<img src="/images/users_embedded.png" alt="Users embedded" />
</Frame>

Click the Edit icon to edit the user’s name.

Click **Add** and enter the user’s email address to invite a new user to Embedded Cloud.

Click a child account to view the Embedded Cloud navigation menu and dashboard:

<Frame>
<img src="/images/embedded_cloud_dashboard.png" alt="Embedded Cloud dashboard" />
</Frame>

From the dashboard, click **Manage Accounts** in the drop-down list to return to the list of child accounts.

<Frame>
<img src="/images/embedded_manage_accounts.png" alt="Embedded manage accounts" />
</Frame>

Click the name of a section in the list below for more details about that section of Embedded Cloud:

- **[Dashboard](/ja/Dashboard-Embedded)** provides a concise view of information for connections and queries.
- **[Sources](/ja/Sources-Embedded)** contains controls for adding and managing data sources.
- **[Logs](/ja/Logs-Embedded)** contains detailed logs for your Embedded Cloud account.
- **[API](/ja/API)** provides documentation for connecting to data via the REST API.
