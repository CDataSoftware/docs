## セットアップガイド

以下の手順で、Dataverse をConnect AI アカウントに接続します。
<Steps>
<Step>
Connect AI のダッシュボードから**Connections** ページを開きます。
</Step>
<Step>
右上の **+ Add Connection** をクリックします。
</Step>
<Step>
検索フィールドに *Dataverse* と入力し、データソース名をクリックします。
</Step>
<Step>
新しいコネクションの**Basic Settings** タブで、コネクション名を入力するかデフォルトの名前をそのまま使用します。
</Step>
<Step>
Enter the **Organization URL** to your organization’s Dataverse data (for example, `https://abc123.crm.dynamics.com/`).
</Step>
<Step>
Select the Authentication method, then proceed to the relevant section and follow those instructions.
</Step>
</Steps>

## Authentication Methods


<Tabs>
  <Tab title="AzureAD">
<Steps>
<Step>
Select a **Schema** from the drop down. System is used for optimal performance, while Entities provides more user-friendly table names.
</Step>
<Step>
OAuth でセキュアに接続するには、**Sign in** をクリックします。これにより、Dataverse のサインインページが新しいタブで開かれます。
</Step>
<Step>
Dataverse アカウントにログインし、リクエストされた権限を設定します（該当する場合）。
</Step>
<SaveAndTest/>
</Steps>
  </Tab>

  <Tab title="AzureServicePrincipal">
<Steps>
<Step>
Enter the **Azure Tenant** that you will use to access data.
</Step>
<Step>
Enter your **OAuth Client Id**.
</Step>
<Step>
Enter your **OAuth Client Secret**.
</Step>
<Step>
Enter your **OAuth JWT Cert**. This is a multi-line key.
</Step>
<Step>
Select your **OAuth JWT Cert Type** from the drop down.
</Step>
<Step>
Select a **Schema** from the drop down. System is used for optimal performance, while Entities provides more user-friendly table names.
</Step>
