## Prerequisites

Before you can connect to Connect AI, you must create OAuth client credentials in TSheets.

<Steps>
<Step>
Log in to TSheets.
</Step>
<Step>
At the left navigation pane, select **Feature Add-ons** > **Manage Add-ons**.
</Step>
<Step>
Select **API Add-On**.
</Step>
<Step>
Click **Install**.
</Step>
<Step>
Select **Add a new application** at bottom left. If a followup window does not display, return to the left navigation pane's **Feature Add-ons** area and select **API**.
</Step>
<Step>
Enter a short name and a meaningful description for the new application.
</Step>
<Step>
Set the **Redirect URL** to *`https://oauth.cdata.com/oauth/`*. This is also called the **Callback URL**.
</Step>
<Step>
Click **Save**. The admin tool generates the OAuth Client Id and OAuth Client Secret that you enter below.
</Step>
</Steps>

## セットアップガイド

以下の手順で、TSheets をConnect AI アカウントに接続します。

<Steps>
<Step>
Connect AI のダッシュボードから**Connections** ページを開きます。
</Step>
<Step>
右上の **+ Add Connection** をクリックします。
</Step>
<Step>
検索フィールドに *TSheets* と入力し、データソース名をクリックします。
</Step>
<Step>
新しいコネクションの**Basic Settings** タブで、コネクション名を入力するかデフォルトの名前をそのまま使用します。
</Step>
<Step>
Enter the **OAuth Client Id** that you obtained in the prerequisites.
</Step>
<Step>
Enter the **OAuth Client Secret** that you obtained in the prerequisites.
</Step>
<Step>
OAuth でセキュアに接続するには、**Sign in** をクリックします。これにより、TSheets のサインインページが新しいタブで開かれます。
</Step>
<Step>
TSheets アカウントにログインし、リクエストされた権限を設定します（該当する場合）。
</Step>
<SaveAndTest/>
</Steps>
