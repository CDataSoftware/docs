## Setup Guide

Follow these steps to connect Jira Assets to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *Jira Assets* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
Enter the **URL** to your Jira Assets endpoint, for example, `https://yoursitename.atlassian.net`.
</Step>
<Step>
Enter the Jira Assets username for authentication in the **User** field.
</Step>
<Step>
Enter the Jira **API token** for the username that you entered in the **User** field. To create an API token:
	- Navigate to [the API Tokens page of your Atlassian account](https://id.atlassian.com/manage/api-tokens).
	- Click **Create API token**.
	- Name your API token and adjust the expiration date, if necessary.
	- Click **Create**.
	- Copy and paste the API token.
</Step>
