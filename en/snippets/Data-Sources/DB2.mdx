## Setup Guide

Follow these steps to connect DB2 to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *DB2* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
Enter the name of the **Server** running DB2.
</Step>
<Step>
Enter the **Port** the DB2 is listening on.
</Step>
<Step>
Enter the name of the DB2 **Database**.
</Step>
<Step>
(Optional) Select whether you want to **Use SSL**.
</Step>
<Step>
Select an **AuthScheme** and follow the relevant instructions below.
</Step>
</Steps>

## Authentication Methods

<Tabs>
  <Tab title="USRIDPWD">
<Steps>
<Step>
To authenticate with user credentials, enter the following information:
	- **User**—enter the username of a user with access to the database.
	- **Password**—enter the password of a user with access to the database.
</Step>

	- **User**—enter the username of a user with access to the database.
	- **Password**—enter the password of a user with access to the database.
</Step>

  <Tab title="IBMIAMAuth">
<Steps>
<Step>
To authenticate using the API key of an application that connects to it, select **IBMIAMAuth** and enter the following information:
	- **User**—enter the the IBM ID or service ID of a DB2 server user.
	- **Password**—enter the API key associated with the application that requires access to the DB2 database.
</Step>
