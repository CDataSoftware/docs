## Setup Guide

Follow these steps to connect Active Directory to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *Active Directory* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
Enter the host name or IP address of the Active Directory server into the **Server** field.
</Step>
<Step>
Enter the port for connecting to the Active Directory server in the **Port** field.
</Step>
<Step>
Enter the Active Directory username for authentication into **User**.
</Step>
<Step>
Enter the user password.
</Step>
<Step>
Enter the **Base DN**. Specifying a narrow base distinguished name limits the scope of searches and can increase performance.
</Step>

</Steps>
