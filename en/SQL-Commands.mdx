## INSERT Statements

## UPDATE Statements

The primary key Id is required to update a record. See [UPDATE Statements](/en/UPDATE) for an example.

## UPSERT Statements

An UPSERT statement updates a record if it exists and inserts the record if it does not. See [UPSERT Statements](/en/UPSERT) for an example.

## DELETE Statements

The primary key Id is required to delete a record. See [DELETE Statements](/en/DELETE) for a syntax reference and example.

## GETDELETED Statements

The GETDELETED query retrieves deleted records. See [GETDELETED Statements](/en/GETDELETED) for an example.

## EXECUTE Statements

Use EXECUTE or EXEC statements to execute stored procedures. See [EXECUTE Statements](/en/EXECUTE) for a syntax reference and examples.

## INSERT INTO SELECT Statements

An INSERT INTO SELECT statement inserts data from a temporary table. See [INSERT INTO SELECT Statements](/en/INSERT-INTO-SELECT) for a syntax reference and examples.

## UPDATE SELECT Statements

UPDATE SELECT statements perform multiple updates in a single request. See [UPDATE SELECT Statements](/en/UPDATE-SELECT) for a syntax reference and examples.

## DELETE SELECT Statements

A DELETE SELECT statement performs multiple deletes in a single request. See [DELETE SELECT Statements](/en/DELETE-SELECT) for a syntax reference and examples.
