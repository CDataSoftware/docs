## Prerequisites

In SAP HANA, add the Connect AI static IP addresses to your connection whitelist.

## Setup Guide

Follow these steps to connect SAP HANA to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *SAP HANA* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
Select an **AuthScheme** and follow the relevant instructions below.
</Step>
</Steps>

## Authentication Methods

<Tabs>
  <Tab title="Password">
<Steps>
<Step>
Enter the following information:
	- **User**—the SAP HANA username for authentication
	- **Password**—the user password
	- **Database**—the name of the SAP HANA database
	- **Port**—the port for connecting to the SAP HANA server
	- **Server**—the host name or IP address of the server running the SAP HANA database
	- **Schema**—(Optional) the SAP database schema
</Step>
