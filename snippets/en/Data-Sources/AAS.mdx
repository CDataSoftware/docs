## Setup Guide

Follow these steps to connect Azure Analysis Services to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *Azure Analysis Services* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
In the **URL** field, enter the URL of the Azure server instance. For example, `asazure://eastus.asazure.windows.net/serverCode`.
</Step>
<Step>
Select the Authentication method, then proceed to the relevant section and follow those instructions.
</Step>
</Steps>

## Authentication Methods

<Tabs>
  <Tab title="AzureAD">
<Steps>
<Step>
Enter the **Azure Tenant** Id found on the Overview page of the OAuth app used to authenticate to Microsoft SharePoint on Azure.
</Step>
<Step>
Click **Sign in** to connect securely through OAuth. This action opens the Azure Analysis Services sign-in page in a new tab.
</Step>
<Step>
Log in to your Azure Analysis Services account and provide the requested permissions (if applicable).
</Step>

</Steps>
  </Tab>

  <Tab title="Azure Password">
<Steps>
<Step>
In **User** enter the Azure Analysis Services username for authentication.
</Step>
<Step>
In **Password** enter the user password.
</Step>
<Step>
Enter the **Azure Tenant** Id found on the Overview page of the OAuth app used to authenticate to Microsoft SharePoint on Azure.
</Step>

</Steps>
  </Tab>

  <Tab title="AzureServicePrincipal">
<Steps>
<Step>
Enter the **Azure Tenant** that you will use to access data.
</Step>
<Step>
Enter your **OAuth Client Id**.
</Step>
<Step>
Enter your **OAuth JWT Cert** OR your **OAuth Client Secret**.
