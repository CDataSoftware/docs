## Prerequisites

Please contact your SAP Fieldglass representative to obtain an Environment URL, OAuth Client Id, and OAuth Client Secret.

## Setup Guide

Follow these steps to connect SAP Fieldglass to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *SAP Fieldglass* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
Enter the following information:
	- **Environment URL**–the instance where SAP Fieldglass is located, for example, `https://myinstance.com`.
	- **OAuth Client Id**–the client Id assigned to your custom OAuth application.
	- **OAuth Client Secret**–the confidential client secret assigned to your custom OAuth Application.
	- **Connector Data Format**–the data format for downloaded data: CSV or JSON.
	- **APIKey**–(optional) the API key used to authenticate to SAP Fieldglass. If you encounter an error indicating that an API key is required, you can request one from SAP Fieldglass administration or log in to the SAP Fieldglass Integration Tools and click **Create API Application**.
</Step>

</Steps>
