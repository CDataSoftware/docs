## Prerequisites

Before you can connect to Connect AI, you must create OAuth client credentials in TSheets.

<Steps>
<Step>
Log in to TSheets.
</Step>
<Step>
At the left navigation pane, select **Feature Add-ons** > **Manage Add-ons**.
</Step>
<Step>
Select **API Add-On**.
</Step>
<Step>
Click **Install**.
</Step>
<Step>
Select **Add a new application** at bottom left. If a followup window does not display, return to the left navigation pane's **Feature Add-ons** area and select **API**.
</Step>
<Step>
Enter a short name and a meaningful description for the new application.
</Step>
<Step>
Set the **Redirect URL** to *https://oauth.cdata.com/oauth/*. This is also called the **Callback URL**.
</Step>
<Step>
Click **Save**. The admin tool generates the OAuth Client Id and OAuth Client Secret that you enter below.
</Step>
</Steps>

## Setup Guide

Follow these steps to connect TSheets to your Connect AI account:

<Steps>
<Step>
Open the **Connections** page of the Connect AI dashboard.
</Step>
<Step>
Click **\+ Add Connection** in the upper-right corner.
</Step>
<Step>
Type *TSheets* into the search field, then click the data source name.
</Step>
<Step>
On the **Basic Settings** tab of the new connection, enter a connection name or keep the default name.
</Step>
<Step>
Enter the **OAuth Client Id** that you obtained in the prerequisites.
</Step>
<Step>
Enter the **OAuth Client Secret** that you obtained in the prerequisites.
</Step>
<Step>
Click **Sign in** to connect securely through OAuth. This action opens the TSheets sign-in page in a new tab.
</Step>
<Step>
Log in to your TSheets account and provide the requested permissions (if applicable).
</Step>

</Steps>
